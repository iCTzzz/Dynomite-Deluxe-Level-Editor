# Dynomite Deluxe Level Editor (Recovered)

This repository contains a working copy of a previously lost level editor for Dynomite Deluxe, along with instructions to enable custom levels ("Stomped Puzzles") in the current Steam version of the game.

---

##  Overview

Dynomite Deluxe still includes hidden support for custom levels, but the official editor was never distributed with modern versions of the game.

This project restores that functionality.

---

##  Features

* Create custom levels using the original editor
* Load custom puzzles in the Steam version
* Revives the hidden "Stomped Puzzles" system

---

##  How to Use

### 1. Prepare the game

Go to your Dynomite Deluxe installation folder and delete:

```
DEFAULT.EGG
```

This forces the game to enter custom level mode.

---

### 2. Create a level

Using the editor:

* Create a new level
*  IMPORTANT: Add multiple difficulties (e.g. Easy, Medium, Hard)

> Single-difficulty levels will NOT be recognized by the game.

---

### 3. Export the level

Save your level. The editor will generate a `.egg` file.

---

### 4. Install the level

Place your `.egg` file inside:

```
/Stomped/
```

---

### 5. Launch the game

* The game will enter the **Stomped Puzzles** menu
* Your custom levels should now appear

---

##  Troubleshooting

### "No Puzzles!"

* Your `.egg` file is invalid
* Make sure it includes multiple difficulties

---

### Game does not enter custom mode

* Ensure `DEFAULT.EGG` has been deleted

---

### Level not showing

* Verify the file is inside `/Stomped/`
* Check the file structure (must be generated by the editor)

---

##  Technical Notes

* `.egg` files are containers, not single levels
* The game validates internal structure before loading
* Custom levels are ignored if they do not meet expected format
